Download ZoneMinder

ZoneMinder comes in ready made packages for many linux distributions. Take your pick:

Ubuntu

RedHat, CentOS,Fedora

Debian

Gentoo

Ubuntu

If you are an Ubuntu user, Isaac Connor is maintaining easy to install ZoneMinder packages. The packages included in the default Ubuntu repos are out of date, so please don't install ZoneMinder using Ubuntu's repos. Installation instructions can be found here. Also, Bill Bunge maintains a useful wiki for Ubuntu that covers other situations that the official guide may not. You can read his wiki here.

Redhat, Fedora and CentOS

If you are a RHEL, Fedora, CentOS or SL user, Andrew Bauer is maintaining zmrepo with an up-to-date ZoneMinder package. Head over to zmrepo for instructions.

Debian

Bill Bunge maintains an updated Debian installation wiki which makes it very easy to install ZoneMinder on Debian. See here.

Gentoo

If you are a Gentoo user, ZoneMinder is available just by typing emerge zoneminder. Thanks to the Gentoo web-apps team for making this available.

Build from source

If you can't a ready to run package for your system, or maybe you just prefer to build everything on your own, ZoneMinder uses the cmake system to build from source. See here.
